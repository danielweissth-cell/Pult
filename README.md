# 📡 Arduino Bluetooth → IR Remote

Проєкт дозволяє керувати інфрачервоним (IR) пристроєм через Bluetooth.  
Команди надсилаються з телефону або іншого BT-пристрою та перетворюються в IR-сигнали.

---

## ⚙️ Можливості

- Керування через Bluetooth (HC-05 / HC-06)
- Відправка IR-команд (NEC протокол)
- Просте налаштування під свій пульт
- Можливість змінювати кнопки та коди

---

## 🧰 Необхідні компоненти

- Arduino (Uno / Nano / інша)
- Bluetooth модуль (HC-05 / HC-06)
- IR світлодіод

---

## 📚 Залежності

⚠️ **ВАЖЛИВО**

Потрібна бібліотека:
- **IRremote версії 2.6.0**

Новіші версії (3.x+) НЕ сумісні з цим кодом.

---

## 🔌 Підключення

### Bluetooth
- RX модуля → Pin 5 Arduino  
- TX модуля → Pin 4 Arduino  
- Живлення → 5V / GND  

> Використовується SoftwareSerial для зв’язку з Bluetooth.

---

### IR
- IR світлодіод → Pin 3 Arduino 
- Другий контакт → GND  

---

## 🚀 Як працює

Arduino отримує байти через Bluetooth і порівнює їх із заданими значеннями.  
Якщо значення співпадає — відправляється відповідна IR-команда.

---

## 🧩 Як змінити кнопки

Ти можеш повністю налаштувати керування під себе:

### 1. Зміна кнопки Bluetooth
- Кожна кнопка — це ASCII-код (число)
- Просто заміни значення на потрібне

Приклад відповідностей:
- `50` → кнопка "2"
- `43` → "+"
- `45` → "-"
- `119` → "w"

---

### 2. Зміна IR-команди
- Замінюється HEX-код (наприклад: `0x20DF22DD`)
- Довжина сигналу зазвичай залишається `32` (для NEC)

---

2. Підключи IR-приймач до Arduino  
3. Відкрий Serial Monitor

5. Копіюй отримані HEX-коди

## 🧪 Відладка

- Всі дії виводяться в Serial Monitor  
- Якщо команда не розпізнана — з’явиться повідомлення  
  **"такого нету"**

## ⚠️ Нюанси

- Використовується SoftwareSerial → стабільно працює на 9600 бод  
- Пін 3 зарезервований під IR  
- Працює тільки з бібліотекою **IRremote 2.6.0**  
- Нові версії бібліотеки можуть викликати помилки
